# 💫 About Me:
Hi, I'm Muhammed Suhaib, a Junior Software Developer / MERN Stack Developer based in Kerala, India. I currently work at **Rootsys International**, where I build core modules — POS, Dashboard, Kitchen Management, Inventory, and more — for **Brixpos**, an AI-powered restaurant management system, and integrate AI (OpenAI, Gemini) and WhatsApp APIs to automate workflows.

I'm also the **Founder & Lead Developer of Moshopo**, a subscription-based, multi-tenant SaaS platform that helps small retail businesses manage shop operations end-to-end — from product catalog and POS to order fulfillment, payments, and custom storefront domains.

I love working across the full MERN stack, with a particular interest in backend architecture, scalable system design, and shipping polished, production-ready products. Always learning, always building.
